# **Tank War**

[![Build Status](https://travis-ci.com/sdwqdwd5/tankWar.svg?branch=master)](https://travis-ci.com/sdwqdwd5/tankWar)

* Implemented JDK13, IDEA 2019.1, MAVEN
* Continuous Integrate using Travis CI
